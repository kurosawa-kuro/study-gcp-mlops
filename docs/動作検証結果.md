#### Run 7 の教訓 (PDCA サイクル効率化)

- **シェル展開依存は原則避ける** — inline python を `sh -c '...'` に書くと escape で必ず事故る。専用 image / 専用 entrypoint script が唯一安全
- **`.dockerignore` は事故要因ランキング最上位** — 包括パターン 1 行で Python package や Dockerfile 依存ファイルを殺すため、追加時は必ず Dockerfile `COPY` 行を grep してテストする
- **IAM grant を要する経路は計画時に除外** — auto mode では IAM grant が harness 拒否されるパターンが多いため、事前に impersonation / TokenCreator / Invoker 依存が無い経路を優先する
