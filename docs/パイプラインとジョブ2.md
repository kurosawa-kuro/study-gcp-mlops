## importルール（これだけ守れば崩れない）

- `domain` -> どこにも依存しない
- `application` -> `domain`, `ports` のみ依存
- `ports` -> Python標準 + 型定義のみ
- `adapters` -> `ports` を実装、外部SDK利用可
- `app` / `pipeline` -> `bootstrap` と `adapters/inbound` のみ呼ぶ
