詳細仕様は [`01_仕様と設計.md`](01_仕様と設計.md)。要点:

### 2.1 Phase 4 → Phase 5 で増える技術 (Phase 7 配下に本実装)

- Vertex AI Pipelines (KFP v2)
- Vertex AI Endpoints (Phase 7 では KServe に差し替え済み)
- Vertex Vector Search (ME5 ベクトル検索の本番 serving index、Phase 4 の BigQuery `VECTOR_SEARCH` から発展)
- Vertex AI Feature Store (Feature Group / Feature View / Feature Online Store) (training-serving skew 防止)
- Vertex Model Registry (`production` / `staging` alias)
- Vertex AI Model Monitoring v2 (drift 監視)
- orchestration は Phase 4 軽量経路を継続するという理解 (Composer 本線化は Phase 6 論理境界 + Phase 7 本実装)

## 3. 不変ルール (論理 Phase)

- **Phase 5 は独立コードを保守しない**。Phase 7 完成版コードを基準にして技術境界を理解する

- 中核 5 要素 (Meilisearch BM25 + Vertex Vector Search + multilingual-e5 + RRF + LightGBM LambdaRank) は Phase 3-7 で不変
