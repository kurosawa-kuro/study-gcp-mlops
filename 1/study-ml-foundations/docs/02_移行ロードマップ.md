> **現状**: 本 phase の実装データ (コード) は 2026-05-02 に削除済 (`pre-phase1-impl-removal` tag で復元可)。改訂方針に基づき、コード化対象として再構築の対象。本 doc は仕様/教育資料として残置。

## 2. Phase 1 のスコープ

詳細仕様は [`01_仕様と設計.md`](01_仕様と設計.md)。要点:

- **題材**: California Housing 回帰 (不変)
- **主眼**: preprocess / feature engineering / training / evaluation / artifact 出力
- **学習器**: LightGBM 回帰 (不変)
- **評価指標**: RMSE / R² (numpy で自前実装)
- **実行基盤**: Docker Compose + PostgreSQL

## 3. 不変ルール

- 題材は California Housing 回帰 (変えない)
- 学習器は LightGBM 回帰 (変えない)
- 評価は RMSE / R² (変えない)
- 成果物は `run_id` / `model.lgb` / `metrics.json` を残す
- API や設計パターンは持ち込まない (Phase 2 以降の責務)
- Phase 2 とコード共有 import は行わず、phase 独立を優先
