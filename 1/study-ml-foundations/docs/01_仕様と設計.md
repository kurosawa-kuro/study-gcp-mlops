# ML パイプライン設計書

## 概要

カリフォルニア住宅価格予測の学習から推論までをローカルで一貫実行する構成。

## 現行アーキテクチャ

- app: API とプレゼンテーション
- ml/common: 共通設定・ロギング・スキーマ
- ml/data: データ取得・前処理・特徴量生成
- ml/training: 学習
- ml/evaluation: 評価・実験ログ
- ml/registry: モデルメタ情報管理
- ml/serving: 推論ロジック
- pipeline: 実行ジョブ

## データフロー

1. pipeline/data_job がデータ投入
2. pipeline/training_job が学習実行
3. ml/evaluation が metrics と W&B を記録
4. app が models/latest を用いて推論

## 設定

主要設定は BaseAppSettings で一元管理。

- env/config/setting.yaml
- env/secret/credential.yaml
- 環境変数上書き

主なキー:

- postgres_host, postgres_port, postgres_db, postgres_user, postgres_password
- model_dir, model_path
- wandb_api_key, wandb_project, wandb_dir

## モデル

- アルゴリズム: LightGBM 回帰
- 評価: RMSE, R2
- 保存: models/<run_id>/model.lgb, metrics.json
- 最新参照: models/latest

## 実験ログ

- 保存先: artifacts/wandb
- offline / online 両対応
