# 04_運用

## 前提

- Docker + Docker Compose
- Python 3.10+

## 基本フロー

```bash
make build
make seed
make train
```

推論 API は Phase 2 (`2/study-ml-app-pipeline/`) で提供。

## 学習ジョブ

学習実行:

```bash
make train
```

主要な処理順:

1. pipeline/data_job で投入済みの学習データを取得
2. ml/data/preprocess で前処理
3. ml/data/feature_engineering で特徴量生成
4. ml/training で LightGBM 学習
5. ml/evaluation で RMSE/R2 算出と記録

学習エントリーポイント:

- python -m pipeline/training_job/main.py 相当
- 実行実体は python -m pipeline.training_job.main

## 生成物

- モデル: ml/registry/artifacts/<run_id>/model.lgb
- メトリクス: ml/registry/artifacts/<run_id>/metrics.json
- 最新モデル参照: ml/registry/artifacts/latest

## クリーンアップ

```bash
make clean
```

削除対象:
