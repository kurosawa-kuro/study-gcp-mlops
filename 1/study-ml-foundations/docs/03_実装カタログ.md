# 実装カタログ

## app

- app/main.py: FastAPI 起動とページ配信
- app/api/predict.py: /predict, /health
- app/services/prediction_service.py: 推論前処理 + 推論実行
- app/schemas/predict.py: 入力スキーマ

## ml/common

- ml/common/config/base.py: BaseAppSettings
- ml/common/logging/logger.py: get_logger
- ml/common/utils/schema.py: FEATURE_COLS, MODEL_COLS, TARGET_COL
- ml/common/utils/run_id.py: run_id 生成

## ml/data

- ml/data/loaders/config.py: データ接続設定
- ml/data/loaders/repository.py: PostgresRepository
- ml/data/preprocess/preprocess.py: 欠損・外れ値・log 変換
- ml/data/feature_engineering/feature_engineering.py: 特徴量追加

## ml/training

- ml/training/model_builder.py: 学習パラメータ
- ml/training/trainer.py: 学習・評価・保存

## ml/evaluation

- ml/evaluation/config.py: 評価設定（wandb_dir 含む）
- ml/evaluation/metrics/regression.py: RMSE, R2, metrics 保存
- ml/evaluation/report/tracking.py: W&B 初期化・ログ送信

## ml/registry

- ml/registry/model_registry.py: run メタ情報登録
- ml/registry/artifact_store.py: アーティファクトパス管理
- ml/registry/metadata_store.py: メタデータ保存

## ml/serving

- ml/serving/predictor.py: 単体推論
- ml/serving/batch_predictor.py: バッチ推論
- ml/serving/response_builder.py: API 応答生成

## pipeline

- pipeline/data_job/main.py: seed 処理
- pipeline/training_job/main.py: 学習ジョブ
- pipeline/evaluation_job/main.py: latest metrics 確認
- pipeline/batch_serving_job/main.py: バッチ推論ジョブ雛形
- pipeline/workflow/orchestration.md: 実行順メモ

## scripts

- scripts/ml/seed.py: seed サービス起動
- scripts/ml/train.py: trainer サービス起動
- scripts/app/serve.py: API サービス起動
- scripts/test.py: pytest 実行
- scripts/clean.py: ml/registry/artifacts クリーン
